# Twitter Clone with Vue.js

Just a twitter clone maded with Vue.js and Pure CSS.

## 💻 Techs

- Vue.js
- Vue Router
- Vuex
- ...

## 📦 Installation

#### 📰 Clone the project

```bash
git clone https://github.com/kaanersoy/twitter-clone.git
```

#### 🔻 Install Dependencies

```bash
cd <project-path>
# With yarn:
yarn

# With npm:
npm install
```

#### 🏃‍♂️ Start the project

```bash
# With yarn:
yarn serve

# With npm:
npm run serve
```
